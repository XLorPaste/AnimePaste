# 使用 CLI

> 👷‍♂️ 文档内容尚未完全更新到新版。

```text
anime/0.1.0-beta.18

Create your own Anime Space

Usage: anime [OPTIONS] <COMMAND>

Commands:
  anime space                    Display the space directory
  anime run <command> [...args]  Run command in the space directory
  anime watch                    Watch anime system update
  anime refresh                  Refresh the local anime system
  anime introspect               Introspect the local anime system
  anime garden list [keyword]    List videos of anime from AnimeGarden
  anime garden clean             Clean downloaded and animegarden cache
  anime bangumi search <input>   Search anime from bangumi and generate plan
  anime bangumi generate         Generate Plan from your bangumi collections

Options:
  -h, --help     Print help
  -v, --version  Print version
```

## anime space

打开配置目录。

## anime refresh

刷新动画资源。
