# 集成媒体库软件

AnimeSpace 命令行程序的配置方式见 [配置 | AnimeSpace](/admin/)。

所有下载的动画资源默认位于 `~/.animespace/anime/` 目录，你只需要配置使用的媒体库软件扫描该目录即可。AnimeSpace 自动下载的动画资源，大部分都可以直接被媒体库软件自动识别，无须二次刮削；AnimeSpace 同样还支持配置动画名称，季度和文件命名格式，方便软件识别。
